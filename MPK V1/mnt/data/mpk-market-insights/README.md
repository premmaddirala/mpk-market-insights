# MPK Market Insights v1.0

Beginner-friendly signal dashboard for Chrome. This version does not connect to Zerodha and does not place orders. It uses a TradingView chart widget and a demo Hybrid Signal Engine layout.

## Publish online using GitHub + Vercel

1. Create a new GitHub repository named `mpk-market-insights`.
2. Upload all files from this folder to that repository.
3. Go to https://vercel.com and sign in with GitHub.
4. Click Add New Project.
5. Select `mpk-market-insights`.
6. Click Deploy.
7. Open the Vercel website link in Chrome.

## Local testing

```bash
npm install
npm run dev
```
Open: http://localhost:3000

Signals are educational/rule-based and not guaranteed. Always use your own confirmation and risk management before trading.
